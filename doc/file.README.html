mm

www.github.com/andrewcsmith/mm

DESCRIPTION:

Implements some Morphological Metrics, described by Larry Polansky.

FEATURES/PROBLEMS:

  • Works on anything Enumerable

  • Can get a number of different MMs

  • Works extra fast on large NMatrix objects
